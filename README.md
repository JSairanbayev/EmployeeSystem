# EmployeeSystem

Console employee management system written in C++ using inheritance and polymorphism.

## Features

- Add different employee types (Developer, Manager, Intern)
- Each type has unique salary calculation
- View all employees with roles and salaries
- Polymorphic storage in a single collection

## Salary calculation

- Developer: base salary + projects × 5000
- Manager: base salary + 10% of sales
- Intern: base salary only

## Technologies

- C++
- OOP: inheritance, polymorphism, virtual methods, override
- Abstract base class (pure virtual functions)
- Pointers and dynamic memory
- STL (vector)

## How to run

1. Clone the repository
2. Open `EmployeeSystem.cpp` in Visual Studio or any C++ compiler
3. Build and run
